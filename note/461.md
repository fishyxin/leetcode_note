# 461. 汉明距离[简单]

[题目链接](https://leetcode-cn.com/problems/hamming-distance/)

## 解法
按照进制转换算法做
```go
func hammingDistance(x int, y int) int {
    var ans int
    for x != 0 && y != 0 {
        l1, l2 := x%2, y%2
        if l1 != l2 {
            ans = ans + 1
        }
        x, y = x/2, y/2
    }
    for x != 0 {
        l := x % 2
        if l != 0 {
            ans = ans + 1
        }
        x = x / 2
    }
    for y != 0 {
        l := y % 2
        if l != 0 {
            ans = ans + 1
        }
        y = y / 2
    }
    return ans
}
```
